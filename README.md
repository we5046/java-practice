# java-practice

KB IT's Your Life 7기 개인 실행 플랜에 맞춰 Java를 복습하고 실습하는 저장소입니다.

이 저장소는 단순 수업 예제 보관용이 아니라,  
배운 내용을 직접 다시 구현하고, 실습 대비 코드를 정리하고,  
객체지향과 Java 기본기를 확실히 다지기 위한 개인 학습 기록용 저장소입니다.

---

## 목표

- Java 기초를 안정적으로 다지기
- 객체지향 개념을 코드로 다시 구현하기
- 실습/평가 대비 코드를 직접 작성하기
- 이후 알고리즘, DB, 백엔드 학습으로 자연스럽게 연결하기

---

## 현재 학습 우선순위

- 클래스 / 객체 / 생성자
- 상속 / 다형성 / 오버라이딩
- 추상 클래스 / 인터페이스
- 예외 처리
- 컬렉션(List, Set, Map)
- 제너릭 기초
- 파일 입출력 기초
